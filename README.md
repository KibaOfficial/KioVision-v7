### KioVision
KioVision is a modified version of the ClearVision v7 theme for Vencord. It provides a sleek and intuitive user interface with various features including:
- Customizable color schemes
- Enhanced readability
- Smooth animations
- Support for all Discord themes (Light, Ash, Dark, Onyx)
- Responsive design for different screen sizes

## Installing

**For Vencord:**

1. Download [`KioVision_v7.theme.css`](https://raw.githubusercontent.com/KibaOfficial/KioVision-v7/master/KioVision_v7.theme.css)
2. Place the file in the Vencord themes directory: `%appdata%\Vencord\themes`
3. Apply the theme through the Vencord settings.

> The theme automatically loads the required CSS files from GitHub.

For more details, visit the [ClearVision documentation](https://clearvision.github.io/).
